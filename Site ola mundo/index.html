Criação e Gerenciamento de Repositórios no GitHub

Durante minha jornada de aprendizado, adquiri experiência prática com o GitHub, uma plataforma essencial para o controle de versão e colaboração em projetos de desenvolvimento de software. Aqui estão alguns dos conceitos chave que aprendi e apliquei:

Iniciação de um Repositório: Aprendi a iniciar um novo repositório no GitHub, que é o primeiro passo para qualquer projeto. Isso inclui a criação de um README para documentar informações importantes sobre o projeto.

Controle de Versão: Familiarizei-me com os conceitos fundamentais do Git, incluindo commits, branches e merges. Isso me permite fazer alterações no código sem afetar o projeto principal até que eu esteja pronto para integrá-las.

Através do meu apredizado sobre GitHub, estou bem equipada para trabalhar efetivamente em equipe e contribuir para projetos de código aberto.
